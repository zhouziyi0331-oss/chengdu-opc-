## 爬虫功能的特殊说明

### 问题：
爬虫功能可能在某些免费服务上受限：
- 掘金、知乎可能有反爬机制
- GitHub Trending相对稳定

### 解决方案：

**选项1：简化爬虫（推荐）**
只爬取GitHub Trending，放弃掘金和知乎

**选项2：使用RSS订阅**
改用RSS源获取内容，更稳定：
```javascript

- GitHub Trending RSS
- 机器之心RSS
```

**选项3：手动更新**
管理员在后台手动添加AI资讯链接（最稳定）
